# Aesthetic Weather App

Website cuaca sederhana dengan desain modern Glassmorphism dan background yang estetik. Dibuat menggunakan Vanilla JavaScript.

## Fitur Utama
* **Real-time Data:** Mengambil data cuaca langsung dari OpenWeatherMap API.
* **Auto Slideshow:** Background berubah otomatis (Salju, Gugur, Panas).
* **Responsive:** Tampilan rapi di Laptop maupun HP.
* **Search UI:** Transisi animasi halus saat mencari kota.

## Teknologi
* HTML5
* CSS3 (Flexbox & Glassmorphism)
* JavaScript (Fetch API)

## Cara Menjalankan
1.  Download atau Clone repository ini.
2.  Buka file `script.js`.
3.  Masukkan API Key OpenWeatherMap kamu di baris pertama:
    ```javascript
    const apiKey = "MASUKKAN_KEY_DISINI";
    ```
4.  Buka file `index.html` di browser.
